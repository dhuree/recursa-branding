# Identity

> Name, personality, and presentation of the Recursa brand

## Name

**Recursa** (reh-KUR-sah)

From Latin *recursus*, meaning "a running back" or "return." The name embodies the core concept: systems that loop back to improve themselves.

### Name Usage

| Context | Format |
|---------|--------|
| Primary brand | Recursa |
| Repository | recursa |
| Package name | recursa |
| Domain | recursa.net |
| Social handles | @recursa |
| Formal reference | The Recursa Framework |

### What We Are NOT Called

- RecursA (mixed case)
- RECURSA (all caps, except in logos)
- Recursa.js/Recursa.py (we're language-agnostic)
- The Recursa (except in "The Recursa Framework")

## Personality Traits

### Primary Traits

1. **Curious** (30%)
   - Asks good questions
   - Explores deeply
   - Finds joy in understanding

2. **Precise** (25%)
   - Values clarity
   - Speaks accurately
   - Documents thoroughly

3. **Witty** (20%)
   - Finds humor in recursion
   - Uses clever wordplay
   - Makes complex ideas fun
   - Self-aware, self-referential humor

4. **Warm** (15%)
   - Welcomes newcomers
   - Celebrates contributions
   - Assumes good intent

5. **Ambitious** (10%)
   - Thinks big
   - Pushes boundaries
   - Believes in potential
